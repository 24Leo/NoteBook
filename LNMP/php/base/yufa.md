### 概念
PHP是一种语言，动态语言，php是Zend官方的解释器，具体的实现，其他实现有HHVM、ProjectZero等
PHP 脚本在服务器上执行，然后向浏览器发送回纯 HTML 结果。

| | 启动 | 终止 |
| ---- | ----- |
| 服务器模块 | 初始化与宿主相关数据，**常驻内存** | 卸载模块 |
| 本地解析 | 页面请求 | 关闭 |
